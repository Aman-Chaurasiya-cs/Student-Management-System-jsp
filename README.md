# Student Management System

A web-based Student Management System developed using JSP, Servlet,JDBC and MySQL.

## 🚀 Features
- Add Student Name/Data 
- Update Student Data
- Delete Student Name/Data 
- Search Student
- Login System
- Register System

## 🛠 Tech Stack
- Java
- JSP
- Servlet
- JDBC
- MySQL
- HTML, CSS

## 📌 Description
This project demonstrates CRUD operations and database connectivity using JDBC.
